# Zoo Labs Foundation

Open AI research network for decentralized training, inference, and scientific computing. A 501(c)(3) nonprofit advancing decentralized AI (DeAI) and decentralized science (DeSci) through community-driven research coordination and open infrastructure.

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Zoo Labs Foundation operates at the intersection of AI and decentralized systems. Through ZIPs (Zoo Improvement Proposals) governance, the community coordinates bleeding-edge research in distributed training, federated inference, and open scientific computing. All research outputs are open-access.

## Quick Start

```bash
# Clone the Zoo ecosystem
git clone https://github.com/zoo-labs/zoo
cd zoo
npm install
npm run dev
```
